# todo-fullstack-app

A simple and modern To-Do List application built using React. This app helps users manage daily tasks efficiently with features like adding, deleting, and marking tasks as completed.

The application uses browser localStorage to store data, ensuring tasks remain saved even after refreshing the page.

The project includes:

Frontend (React) for UI
Backend (Express.js) for API handling
localStorage (current version) for data persistence

🚀 Features
➕ Add new tasks
❌ Delete tasks
<!-- ✅ Mark tasks as completed -->
💾 Persistent storage using localStorage
🎨 Clean and responsive UI
🔄 Real-time updates

🛠️ Tech Stack
    Frontend
        React (Vite)
        React Hooks (useState, useEffect)
        CSS (Flexbox)
    Backend
        Node.js
        Express.js
        REST API
